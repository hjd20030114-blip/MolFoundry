# PRRSV抑制剂设计平台Web界面使用指南

## 🌐 访问地址
**本地访问**: http://localhost:8501  
**网络访问**: http://172.20.10.2:8501  
**外部访问**: http://38.207.136.90:8501

## 🚀 快速开始

### 1. 打开Web界面
在浏览器中访问 http://localhost:8501，您将看到PRRSV抑制剂设计统一平台的主界面。

### 2. 开始新运行
点击右上角的 **"🚀 开始新运行"** 按钮，系统会创建一个新的运行目录来保存所有结果。

## 📋 功能模块详解

### 🏠 项目概览
- **功能**: 查看平台介绍和统计信息
- **内容**: 项目目标、核心功能、最新结果统计
- **操作**: 浏览平台信息，查看历史结果

### 🧪 配体生成
- **功能**: 智能生成候选小分子化合物
- **步骤**:
  1. 设置生成参数（分子数量、生成方法）
  2. 选择生成方法：
     - 优化生成（推荐）
     - 随机生成
     - CMD-GEN生成（深度学习）
  3. 点击 **"🚀 开始生成"**
- **结果**: 生成高质量配体分子，显示分子信息表格和性质分布图

### 🔬 分子对接
- **功能**: 使用AutoDock Vina进行高精度分子对接
- **步骤**:
  1. 选择受体文件（推荐使用1p65.pdbqt）
  2. 设置对接参数（搜索精度、构象数）
  3. 点击 **"🚀 开始对接"**
- **结果**: 显示结合亲和力分布、对接质量统计、前10名最佳结果

### 📈 ADMET分析
- **功能**: 全面评估分子的药物类似性质
- **步骤**:
  1. 选择分析类型（Lipinski规则、分子性质、毒性预测等）
  2. 点击 **"🚀 开始分析"**
- **结果**: 显示Lipinski符合率、分子性质散点图、ADMET评估结果

### 📊 结果分析
- **功能**: 综合分析所有结果
- **内容**: 统计信息、前10名最佳化合物、综合评分
- **操作**: 自动整合对接和ADMET结果

### 🖼️ 2D分子图像
- **功能**: 生成高质量2D分子结构图像
- **步骤**:
  1. 设置显示参数（分子数量、图像质量）
  2. 配置网格设置（列数、行数）
  3. 点击 **"🎨 动态生成2D分子图像"**
- **结果**: 生成单个分子图像和网格图，提供下载选项

### 🌐 3D可视化
- **功能**: 交互式3D分子结构展示
- **选项**:
  - **生成完整3D报告**: 一键生成所有3D可视化
  - **单个分子3D结构**: 查看特定分子的3D结构
  - **蛋白质-配体复合物**: 展示结合模式
  - **交互式数据仪表板**: 数据分析图表
- **结果**: 生成HTML格式的3D可视化文件

### 🚀 完整工作流程
- **功能**: 一键式完整药物设计流程
- **步骤**:
  1. 设置流程参数（分子数量、受体文件、是否包含3D可视化）
  2. 点击 **"🚀 启动完整工作流程"**
- **流程**: 配体生成 → 分子对接 → ADMET分析 → 3D可视化
- **结果**: 完整的分析报告和可视化文件

### 🤖 深度学习流水线
- **功能**: AI驱动的分子设计和优化
- **阶段**:
  - 第一阶段：SE(3)-Equivariant GNN评分器
  - 第二阶段：Pocket-conditioned Transformer生成
  - 第三阶段：Diffusion模型和强化学习优化
- **操作**: 上传蛋白质文件，设置生成目标，选择执行阶段

## 🎯 推荐使用流程

### 新手用户
1. **开始新运行** → 创建运行目录
2. **配体生成** → 生成候选分子
3. **分子对接** → 预测结合模式
4. **ADMET分析** → 评估药物性质
5. **结果分析** → 查看综合结果
6. **3D可视化** → 生成可视化报告

### 高级用户
1. **完整工作流程** → 一键执行所有步骤
2. **深度学习流水线** → AI驱动设计
3. **2D分子图像** → 生成高质量图像
4. **文件管理** → 管理结果文件

## 💡 使用技巧

### 参数设置建议
- **分子数量**: 新手建议10-50个，高级用户可设置100-200个
- **搜索精度**: 8-16（平衡速度和精度）
- **生成构象数**: 5-20个（通常9个足够）
- **图像质量**: 高质量（平衡效果和速度）

### 结果查看
- **实时更新**: 所有结果都会实时显示在界面上
- **下载功能**: 支持CSV、HTML、PNG等格式下载
- **3D查看**: 点击"直接打开"按钮在浏览器中查看3D结构

### 故障排除
- **如果Web界面无法访问**: 检查端口8501是否被占用
- **如果对接失败**: 确保受体文件存在且格式正确
- **如果3D可视化不显示**: 检查浏览器是否支持JavaScript

## 🔧 技术说明

### 系统要求
- **浏览器**: Chrome、Firefox、Safari、Edge（推荐Chrome）
- **网络**: 本地网络访问
- **JavaScript**: 必须启用

### 性能优化
- **分子数量**: 建议不超过200个分子
- **3D可视化**: 大量分子时可能需要较长生成时间
- **浏览器缓存**: 定期清理浏览器缓存

## 📞 技术支持

如果遇到问题，请检查：
1. 浏览器控制台是否有错误信息
2. 网络连接是否正常
3. 系统资源是否充足
